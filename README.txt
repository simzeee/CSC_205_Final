HOW TO COMPILE AND RUN
================================================================================

COMPILATION:

Navigate in your terminal to the project directory and compile all Java files:

    javac *.java

RUNNING THE PROGRAM:

    java Main

================================================================================
PROJECT DESCRIPTION
================================================================================

This project implements three data structure applications:

1. Suitor Selection - Uses a circular linked list to simulate the Josephus
   problem

2. Haunted House - Implements a graph structure for navigating through
   connected rooms

3. Sentiment Analysis - Uses HashSets to analyze text files for positive
   and negative sentiment
